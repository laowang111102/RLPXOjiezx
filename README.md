# 多线程多人聊天室系统 python1003

## 项目概述

本项目是一款基于 Python 编写的多线程多人聊天室系统。通过运用 Socket 编程和 tkinter 组件，实现了用户之间的即时通讯功能。

## 技术栈

- 编程语言：Python
- 网络通信：Socket 编程
- 图形界面：tkinter 组件

## 功能模块

1. 用户注册与登录
2. 创建和加入聊天室
3. 实时消息发送与接收
4. 查看在线用户列表
5. 私聊功能
6. 聊天室管理员权限设置

## 系统角色

- 普通用户：可注册、登录、加入聊天室、发送和接收消息、查看在线用户等。
- 管理员：除普通用户权限外，还可以设置聊天室管理员、禁言用户等。

## 运行环境

- 操作系统：Windows、Linux、macOS
- Python 版本：3.x

## 部署步骤

1. 安装 Python 3.x
2. 克隆项目源码
3. 安装依赖库：tkinter、socket
4. 运行主程序

```
.

```
